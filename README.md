# @amlplugins/google-docs

Native `@googleapis/docs` SDK re-exported under the `@amlplugins` namespace.

> Google Docs (docs_v1).

## Install

```bash
npm install @amlplugins/google-docs
```

## Use

```ts
import { /* named exports from @googleapis/docs */ } from "@amlplugins/google-docs";
// or
import sdk from "@amlplugins/google-docs";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
